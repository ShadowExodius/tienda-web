# logica del negico: Trazo & Piedra

## Descripcion
 
Somos una empresa que trabaja bajo pedido y inventario en ropa con perlas

## flujo Principal
1. El usuario ingresa a el catalogo
2. el sistema verifica si hay stock
3. Si Hay Stock: Stock disponible
4. Si No Hay Stock: Stock no disponible

## Pseudocodigo

Si Stock disponible
    Mostrar "Stock disponible"

Si no:
    Mostrar: "Stock no disponible"
